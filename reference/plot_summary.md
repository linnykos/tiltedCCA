# Plot summary of D-CCA

Plot summary of D-CCA

## Usage

``` r
plot_summary(
  obj,
  xlab = "Latent dimension",
  ylab1 = "CCA objective",
  ylab2 = "Distinct % for Modality 2",
  main = "",
  pch = 16,
  cex = 1,
  lwd = 1,
  ...
)
```

## Arguments

- obj:

  output of `dcca_decomposition`

- xlab:

  `xlab` graphical parameter

- ylab1:

  `ylab` graphical parameter of left axis

- ylab2:

  `ylab` graphical parameter of right axis

- main:

  `main` graphical parameter

- pch:

  `pch` graphical parameter

- cex:

  `cex` graphical parameter

- lwd:

  `lwd` graphical parameter

- ...:

  additional graphical parameter

## Value

shows a plot but returns nothing
