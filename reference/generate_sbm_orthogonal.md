# Generate orthogonal matrices correspond to an SBM

Generate orthogonal matrices correspond to an SBM

## Usage

``` r
generate_sbm_orthogonal(B_mat, membership_vec, centered = T)
```

## Arguments

- B_mat:

  symmetric matrix

- membership_vec:

  vector of positive integers

- centered:

  boolean, where if `TRUE`, one less dimension is provided but the
  singular vectors are centered

## Value

matrix
